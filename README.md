# 🐻 Five Night's at Pedralbes

Benvingut al repositori oficial de **Five Night's at Pedralbes**, un joc 2D multijugador de supervivència i terror desenvolupat com a projecte transversal per al cicle de **2DAM (2025-2026)**.

## 👥 Integrants
*   **Aymar Ramos Aguilar**

## 📝 Descripció del Projecte
Five Night's at Pedralbes és una experiència multijugador en temps real on els jugadors han de sobreviure en un entorn hostil. El projecte posa a prova la sincronització de dades mitjançant WebSockets i una arquitectura de contenidors robusta.

*   **Client:** Desenvolupat en **Unity (C#)** amb UI Toolkit.
*   **Servidor:** API REST i WebSockets amb **Node.js i Socket.io**.
*   **Base de dades:** **MongoDB** per a la persistència d'usuaris i estadístiques.
*   **Infraestructura:** Desplegament amb **Docker** i proxy invers amb **Nginx**.
