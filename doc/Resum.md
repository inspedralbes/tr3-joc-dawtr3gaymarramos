# Resum del Projecte: Five Night's at Pedralbes

**Curs:** 2DAM 2025-2026  
**Integrant:** Aymar Robinson Ramos Aguilar

**Abstract:**  
Joc 2D multijugador de supervivència. El client (Unity/C#) s’encarrega de l’experiència immersiva, mentre que el servidor (Node.js/Socket.io) gestiona la sincronització en temps real i les sales d’espera. Utilitza MongoDB per persistència de dades, Nginx com a proxy invers i Docker per a un desplegament robust i escalable. La interacció es basa en WebSockets per al joc i rutes REST per a la gestió d'usuaris.
